1. Shared Status and Feedback Family

All components strictly utilize the Manrope font for all typographic elements. Surfaces use tonal layering via surface-container values to ensure visual hierarchy.

2. Semantic Mapping Rules

  • Success: Teal (#006A6A) - Affirmative outcomes.
  • Warning: Amber (#E56C3E) - Time-sensitive info.
  • Error: Red (#BA1A1A) - Blocking issues.

3. Urgency and State Transitions

Transient feedback (Toasts) persists for 3.5 seconds. Permanent states (Empty/Error cards) must provide at least one clear primary CTA to ensure user progression.

4. Shared Form Continuity

Inline validation must map 1:1 with the status family. Use error backgrounds and semantic icons to ensure accessibility and clear communication of field states.
