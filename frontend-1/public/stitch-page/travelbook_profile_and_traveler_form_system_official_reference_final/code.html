System Implementation Rules

Field Validation
  • Passport: Regex validation for alphanumeric characters, auto-uppercase on input.
  • Dates: Past dates only for DOB; future dates for Passport Expiry.
  • Real-time inline error styling using border-error/50.
Flow Rules
  • Save/Discard row persists at top-right for desktop and bottom-sticky for mobile.
  • Changes are only applied after explicit Save action.
Modal Behavior
  • Desktop: Centered modal with backdrop-blur.
  • Mobile: Bottom-sheet drawer for form entry.
  • Esc key or background click cancels.
Continuity Notes
  • Traveler IDs link directly to Document Management profiles.
  • Profile updates propagate to pending booking documents.
