edit_note

Copy Tone Principles

Practical

Prioritize clarity over cleverness. Use active verbs to guide the user's journey.

Calm

Reduce travel anxiety through supportive language and predictable outcomes.

Trustworthy

Be transparent about fees, times, and policies. Avoid marketing fluff.

Concise

Respect the traveler's time. If it doesn't add value, remove it.
