Continuity Guidelines

01

Spacing is Luxury

If a layout feels crowded, jump two levels up the spacing scale. White space is functional.

02

No Harsh Borders

Avoid solid 1px dividers. Use background tonal shifts or generous spacing to define content boundaries.

03

Typography First

Pair Manrope's authority with Inter's clarity. Keep metadata small, tracked-out, and bold.
