06

Mobile Behavior Rules

menu TRAVELBOOK

Shell Pivot

Sidebar collapses into a mobile navigation trigger. Layouts stack vertically for optimal readability.

smartphone
Responsive Breakpoints

Mobile (< 768px): Vertical stacking, hidden sidebar, 16px margins.

Desktop (> 1024px): 256px Fixed sidebar, 1200px centered content canvas.

touch_app
Tap Targets

Minimum touch area of 44x44px for all mobile interactions. Forms use full-width buttons on small viewports.

verified Continuity Standards

Typography

Headers must remain Manrope Bold or Extrabold. Body text uses Inter for utility. Minimum contrast ratio of 4.5:1 for all text elements.

Spatial Geometry

24px radius for major containers. 12px for interior elements. Layout sectioning relies on tonal shifts rather than 1px borders.

Visual Depth

Depth is established through tonal layering (bg-surface vs bg-surface-container). Elevation shadows only for floating components.
