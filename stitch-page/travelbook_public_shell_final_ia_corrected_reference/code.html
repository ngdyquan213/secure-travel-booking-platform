SYSTEM ARCHITECTURE

Public Shell Reference

Defining the structural logic, aesthetic standards, and component DNA for the TravelBook brand experience.

01. Public Shell Rules

The shell is our architectural anchor. It maintains brand consistency across all public-facing touchpoints with a focus on serenity and clarity.

The 1200px Constraint

All main content must be contained within a centered 1200px max-width container to ensure optimal line length and visual focus on large displays.

Sticky Identity Anchor

The TopAppBar must remain sticky using a glassmorphism backdrop-blur effect. It must never use a solid border; separation is achieved through a subtle 5% shadow.

04. Mobile Logic

For public-facing pages, we prioritize a clean, editorial aesthetic. Mobile navigation is simplified to a persistent TopAppBar containing only the Brand Identity and a Menu Trigger.
